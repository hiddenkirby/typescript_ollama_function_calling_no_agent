# A simple example of 
1. using ollama (llama3.1)
2. searxng for search, 
3. and simple prompts to showcase function calling (vs using Agents and tools API)

# prereqs
1. nvm
2. ollama installed
3. searxng installed (i installed searxng-docker)

# to run
1. `nvm use`
2. `npm run build`
3. `npm run start`
